# Оптимизация изображений на фронтенде в 2026 году

\## Введение

Изображения до сих пор составляют **60-70%** веса любой современной веб-страницы. В 2026 году, с появлением новых форматов и API, у нас есть все инструменты, чтобы сделать загрузку изображений практически мгновенной.

## \## Новые форматы изображений

\### AVIF — король сжатия

В 2026 году AVIF поддерживается во всех современных браузерах (96% глобальной поддержки). Он дает:

\- **50%** меньший вес по сравнению с JPEG

\- **30%** по сравнению с WebP

\- Поддержка HDR и глубины цвета 10/12 бит

### \### JPEG XL — темная лошадка

JPEG XL набирает популярность благодаря:

*   Обратной совместимости с JPEG
    
*   Лучшему сжатию без потерь
    
*   Поддержке анимации
    

````
```bsl
// next.config.js — настройка форматов
module.exports = {
  images: {
    formats: ['image/avif', 'image/webp', 'image/jxl'],
    remotePatterns: [

        protocol: 'https',

      },
    ],
  },
````
