Терминал и Vim пугают многих разработчиков, но это невероятно мощный инструмент. Рассказываю о своём пути перехода с VS Code на Neovim: плагины для LSP, автодополнение, работа с деревом проекта, отладка и горячие клавиши, которые ускорили мою работу в несколько раз. Спойлер: теперь я не могу вернуться к обычным IDE. Конфиги и полезные ссылки прилагаются.
